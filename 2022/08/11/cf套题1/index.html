A

题目大意：

Cirno给他的学生一个正整数x，并且布置了一项作业，要求他的学生找到最小的正整数

要满足两个条件： x & y > 0 并且 x .xor. y > 0

思路：

有&和.xor.，很明显这是一道涉及位运算的题目，而一般位运算都可以通过找规律来做。所以这题的规律就是，如果x小于3的话，y最小为3；如果x > 3的话，就要考虑从右往左数1在第几位上，如果lowbit(x) == x的话，那么y最小为x + 1，否则y == lowbit(x)

B

题目大意：

Patchouli正在制作一个神奇的护身符。她最初有n个魔法令牌。它们的力量可以用$a_1,a_2,…,a_n$来表示。

Patchouli可以对令牌执行两种操作。

融合：Patchouli选择两个令牌，将两个令牌融合成一个，其魔力等于所选的两个令牌魔力之和。

减少：Patchouli选择一个令牌，将其移除并创建一个新的令牌，其魔力等于 x/2

当它们的魔力是奇数时，令牌会更有效。请帮助Patchouli找出她需要的最少操作次数，以使所有令牌的魔力为奇数。

思路：

我们知道，even + odd = odd，所以只要有一个令牌奇数的就可以了。

如果令牌已经全为奇数魔力值，则答案为0；如果令牌已经全为奇数魔力值，则将最小的一个令牌一直做“减少”操作直到等于1，然后在与其他的所有令牌进行“融合”操作，最小偶数令牌做“减少”操作到1的次数为x，那么答案为 x + n。

如果令牌有odd也有even，那么答案就为even的个数。

C

题目大意：

最初有一个字符c，每次操作可以选定其中一个子串替换成另一个子串，给出最终形态，还有打乱的操作序列，求这个最初字符c

思路：

暴力模拟会tle。

可以考虑单个字符对字符串的贡献。假设s1为t1的所有字符集合，s2为t2所有字符的集合

那么c + s2 = 最终形态 + s1，ch = 最终形态 + s1 - s2

所以除了c，其他字符都会被抵消掉，所以在所有字符的集合之和，出现奇数次的字符就是最初的字符c

D

题目大意：

给定1~n个点，每个点有一个权值a[i]，有一只兔子可以从任意一点出发，每次可以向离当前位置x距离 <=1的y移动，并且得到这一路上的权值，每次走过的点的权值会+1。求k个单位时间后，小兔子能得到的最大权值。

思路：

首先可以想到用前缀和来做。

为了让得到的权值最大，要贪心来做。从任意一个地方出发，可以知道朝着一个方向的走得到的权值是越来越大的。因为一个位置可以重复经过。所以每个位置的贡献为 权值加上时间k - 1

当 k == n时，n个点全部遍历一遍，答案为 sum[n] + k * ( k - 1) / 2

当 k < n 时，取最大的一段长度为k的sum值，然后加上 k * (k - 1) / 2

当 k > n 时，k - 1 + k - 2 + … + k - (n - 1)的总和加上sum[n]

E

题目大意：

一个n个点m条边的无向图，一共可提出$2m$次提问，提问方式为每次输出一个长度为$m$的01字符串s，s[i] = 0表示第$i$条边不存在，s[i] = 1表示第$i$条边存在。系统会给出由当前边所组成的最大森立（边权之和最大的森林），我们需要在$2m$次提问中找出由所有边构成的最小生成森林。

思路：

这是一道交互题。题目是求森林的值，所以一共$m$条边，所以用$m$次提问得到每条边的权值。

创建一个集合s和res记录当前集合所能构成的最小生成树的值，利用$kruskal$的思想，把权值从小到大一次加入集合s中，每加入集合一次后，就进行一次询问，得到一个值x，如果x == 当前边权值 + res，则表示当前边是构成最小生成树的边，然后res = x，如果不等于，则表示当前边不是构成最小生成树的边，则舍去。最后得到的res就是最小生成树的权值。
