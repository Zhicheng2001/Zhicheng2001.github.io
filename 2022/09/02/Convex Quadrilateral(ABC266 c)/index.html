Convex Quadrilateral(ABC266 C)

题目描述：

给一个四边形坐标，判断它四个内角是不是都小c于180°，是则输出Yes，不是则输出No

思路1：

计算出每个内角的角度，考虑到c++中的asin和atan只能表示[−π2,π2][-\frac \pi 2,\frac \pi 2][−2π​,2π​]，所以用atan2来进行计算，它能表示[−π,π][-\pi,\pi][−π,π].

得到所有角度以后只要有一个内角大于180°，则输出No

Code

思路2：

思路1的想法很清晰，计算出所有角度再判断，但是这难免有些麻烦。所以我们可以考虑向量运算的性质。

向量叉乘：

a⃗=(x1,y1),b⃗=(x2,y2)\vec{a} =(x_1,y_1),\vec{b} = (x_2,y_2) a=(x1​,y1​),b=(x2​,y2​)

k=a⃗×b⃗=∣a∣∣b∣sin⁡θ=x1y2−y1x2k = \vec a \times \vec b = |a| |b|\sin \theta = x_1y_2 - y_1x_2 k=a×b=∣a∣∣b∣sinθ=x1​y2​−y1​x2​

当k<0k \lt 0k<0时，sin⁡θ<0\sin \theta < 0sinθ<0，那么aaa正旋转到bbb的角度大于180°

当k>0k \gt 0k>0时，sin⁡θ>0\sin\theta \gt 0sinθ>0，那么aaa正旋转到bbb的角度小于180°

利用这个性质我们也可以AC，并且比思路1更直接省事

由于本题让我对向量有了更深刻的理解，故记录下来
